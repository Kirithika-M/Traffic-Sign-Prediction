# Traffic-Sign-Prediction
## Traffic Sign Classification using CNN

A deep learning project that uses Convolutional Neural Networks (CNN) to classify traffic signs from the German Traffic Sign Recognition Benchmark (GTSRB) dataset.

#### Recommended Python Version for Virtual Environment: 3.10.11

#### Guide to install the modules:
1. Tensorflow (CPU / GPU) 
- Command : pip install tensorflow==2.15.0

2. NumPy
- Command : conda install -c conda-forge numpy=1.24.3

3. Pandas
- Command : conda install -c conda-forge pandas=2.0.3

4. Scikit-learn
- Command : conda install -c conda-forge scikit-learn=1.3.2

5. Matplotlib
- Command : conda install -c conda-forge matplotlib=3.8.0

6. Seaborn
- Command : conda install -c conda-forge seaborn=0.13.0

7. OpenCV
- Command : conda install -c conda-forge opencv=4.8.1

8. Pillow
- Command : conda install -c conda-forge pillow=10.1.0

9. Flask
- Command : pip install Flask==3.0.0

10. Flask-Cors
- Command : pip install flask-cors==4.0.0
